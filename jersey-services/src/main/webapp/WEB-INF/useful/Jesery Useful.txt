--Command to generate maven project
	
mvn archetype:generate -DarchetypeGroupId=org.glassfish.jersey.archetypes -DarchetypeArtifactId=jersey-quickstart-webapp -DarchetypeVersion=2.2

Adding archetypein eclipse;
Open Window > Preferences
Open Maven > Archetypes
Click Add Remote Catalog and add the following:
Catalog File: http://repo1.maven.org/maven2/archetype-catalog.xml
Description: maven catalog
Restart eclipse
